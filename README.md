Título: 📦 Forecasting de Vendas com Facebook Prophet & Power BI

Contexto: O objetivo deste projeto foi prever o faturamento futuro de diferentes categorias do marketplace Olist para otimização de estoque e planejamento financeiro.

Destaques Técnicos:

Time Series: Uso da biblioteca Prophet para capturar sazonalidade.

Grid Search: Implementação de otimização de hiperparâmetros em Python para reduzir o erro (MAPE).

Integração: Pipeline de dados conectando Scripts Python diretamente ao Power Query.

Visualização: Dashboard em Dark Mode com Glassmorphism focado em UX para tomada de decisão.

Resultados: Modelo capaz de prever tendências com intervalos de confiança de 80%, segmentado por categorias de produtos através de ícones dinâmicos.
